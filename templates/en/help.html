When activated, it will scan for the setup users in the given interval.

Using the "New User" Button a new User can be created. You can add multiple MAC addresses in the "MAC Address" field. Atleast one mac address needs to be entered. The plugin will scan for all setup addresses. Once a address was found the state '1' will be send to the miniserver.

The IP address of the FritzBox. Most of the times nothing needs to be changed here.

The TR-064 port of the FritzBox which is used for accessing the MAC addresses. Most of the times nothing needs to be changed here.

The UDP port of the Miniserver the detected Users should be send to.
